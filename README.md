# SwedishLandscapes
A game to select the appropriate landscape on the Swedish map when given a symbol representing that landscape.

# URL to play direct online
https://mctopaz.github.io/SwedishLandscapes/

# Different window sizes
There are many devices with various different window sizes. Make sure to test the layout with for several different sizes.
Here is a list of different sizes:
- SmartPhone portrait: 375x667px (iPhone SE). Supports Nothing Phone(2) and other devices with width less than 600px.
- SmartPhone landscape: Not supported.
- Tablet portrait: 1024x617px (Samsung Galaxy Tab S8) or 1204x588px (Samsung Galaxy Tab S9).
- Tablet landscape: 1024x617px (Samsung Galaxy Tab S8) or 1204x588px (Samsung Galaxy Tab S9).
- Desktop landscape: 1920x911px, 1920x1080px or 1920x1200.
- Desktop portrait: Need fix.
- 4K TVs landscape: 3840x2160px.
- 4K TVs portrait: 3840x2160px. Need fix.
- 8K TVs landscape: 7680x4320px. Very small fonts.
- 8K TVs portrait: 7680x4320px. Need fix.
